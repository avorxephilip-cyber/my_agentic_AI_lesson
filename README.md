# Agentic AI Arcade

A static, interactive web app for learning agentic AI from scratch through a playful curriculum based on the Hugging Face Agents Course.

## What is inside

- Course path based on the official Hugging Face Agents Course units.
- Extra missing concepts: memory, planning, tool contracts, MCP-style protocols, observability, evals, safety, and production guardrails.
- Local browser progress with XP, missions, quiz wins, flashcards, a tool-calling simulator, and a final project checklist.
- No build step and no external dependencies.

## Deploy on GitHub Pages

1. Push these files to a GitHub repository.
2. Open the repository on GitHub.
3. Go to `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the `main` branch and `/root`.
6. Save. GitHub will publish the app after the Pages workflow finishes.
